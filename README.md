# Discourse Virtual Category

Virtual categories let you aggregate topics from other categories based on tag rules. Admins can mark a category as virtual and select tags or tag groups to include in the category topic list. Category permissions are respected, and duplicates are removed.

## Features

- Aggregate topics by tags or tag groups.
- Keep native category topics included.
- Respect Discourse category permissions.
- Admin UI for configuration in category settings.

## Configuration

1. Enable **virtual_category_enabled** in site settings.
2. Edit a category and enable **This is a virtual category**.
3. Choose tags and/or tag groups in the rules modal.
4. Save the category.
